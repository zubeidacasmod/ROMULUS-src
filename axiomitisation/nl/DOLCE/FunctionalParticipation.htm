Natural Language View: Functional Participation

ACTION (class)

Description

If any of the following relationships hold between X and Y then Y is an action: "is used in", "prescribes" and "performs".

If any of the following relationships hold between X and Y then X is an action: "is use of", "is prescribed by" and "is performed by".

ACTIVITY (class)

Description

If any of the following relationships hold between X and Y then Y is an activity: "is result of", "is resource for", "is product of", "is instrument of", "is generic target of" and "is causally follows".

If any of the following relationships hold between X and Y then X is an activity: "is result", "is resource", "is product", "is instrument", "is generic target" and "is causally precedes".

AGENT (class)

Description

If any of the following relationships hold between X and Y then Y is an agent: "is prescribed by" and "is performed by".

If any of the following relationships hold between X and Y then X is an agent: "prescribes" and "performs".

AMOUNT OF MATTER (class)

Description

If X is resource Y then Y is an amount of matter.

If X is resource for Y then X is an amount of matter.

ENDURANT (class)

Description

If any of the following relationships hold between X and Y then Y is an endurant: "is use of", "is target", "is substrate", "is state of", "is product", "is patient", "is generic target" and "is functional participant".

If any of the following relationships hold between X and Y then X is an endurant: "is used in", "is target of", "is substrate of", "is product of", "is patient of", "has as state", "is generic target of" and "is functional participant in".

FOLLOWS (object property)

Description

The property "is causally follows" is a sub-property of "follows".

HAS AS STATE (object property)

Typology

The property "has as state" is a sub-property of "is substrate of".

Description

If X has as state Y then Y is a state.

If X has as state Y then X is an endurant.

X is state of Y if and only if Y has as state X.

INFORMATION OBJECT (class)

Description

If X is theme Y then Y is an information object.

If X is theme of Y then X is an information object.

IS CAUSALLY FOLLOWS (object property)

Typology

The property "is causally follows" is a sub-property of "follows".

Description

The property "is result of" is a sub-property of "is causally follows".

If X is causally follows Y then Y is an activity.

If X is causally follows Y then X is a perdurant.

X is causally follows Y if and only if Y is causally precedes X.

If X is causally precedes Y then X is an activity.

X is causally follows Y if and only if Y is causally precedes X.

IS FUNCTIONAL PARTICIPANT (object property)

Typology

The property "is functional participant" is a sub-property of "is participant".

Description

The following properties are sub-properties of "is functional participant": "is use of", "is substrate", "is product", "is performed by", "is patient" and "is generic target".

If X is functional participant Y then Y is an endurant.

If X is functional participant Y then X is a perdurant.

X is functional participant Y if and only if Y is functional participant in X.
