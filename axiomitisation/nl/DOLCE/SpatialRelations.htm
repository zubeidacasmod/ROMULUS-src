Natural Language View: Spatial Relations

ENDURANT (class)

Description

If any of the following relationships hold between X and Y then Y is an endurant: "is place of", "is participant place", "is descriptive place of" and "is descriptive origin of".

If any of the following relationships hold between X and Y then X is an endurant: "is place", "is participant place of", "is descriptive place" and "is descriptive origin".

IS APPROXIMATE LOCATION (object property)

Typology

The property "is approximate location" is a sub-property of "is generic location".

Description

The following properties are sub-properties of "is approximate location": "is place", "is participant place" and "is descriptive place".

If X is approximate location Y then Y is a spatio temporal particular.

If X is approximate location Y then X is a spatio temporal particular.

X is approximate location Y if and only if Y is approximate location of X.

IS APPROXIMATE LOCATION OF (object property)

Typology

The property "is approximate location of" is a sub-property of "is generic location of".

Description

The following properties are sub-properties of "is approximate location of": "is place of", "is participant place of" and "is descriptive place of".

If X is approximate location of Y then Y is a spatio temporal particular.

If X is approximate location of Y then X is a spatio temporal particular.

X is approximate location Y if and only if Y is approximate location of X.

IS D SPATIAL LOCATION (object property)

Typology
