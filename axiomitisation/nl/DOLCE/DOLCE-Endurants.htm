Natural Language View: DOLCE-Endurants

ABSTRACT (class)

Typology

An abstract is a particular.

Description

An abstract both has as quality only something that is not a temporal location q, and has as quality only something that is not a spatial location q.

Examples

The following are abstracts: sets, regions, and propositions.

Distinctions

No abstract is a quality, or an endurant.

ABSTRACT QUALITY (class)

Typology

An abstract quality is a quality.

Description

An abstract quality both is q location only an abstract region, and has as quality only an abstract quality.

An abstract quality is inherent in a non physical endurant.

Distinctions

No abstract quality is a temporal quality, or a physical quality.

ABSTRACT REGION (class)

Typology

An abstract region is a region.

Description

An abstract region both is q location of only an abstract quality, and is part only an abstract region.

If X is abstract location Y then Y is an abstract region.

If X is abstract location of Y then X is an abstract region.

Distinctions

No abstract region is a temporal region, or a physical region.

AMOUNT OF MATTER (class)

Typology

An amount of matter is a physical endurant.

Distinctions

No amount of matter is a physical object, or a feature.

ARBITRARY SUM (class)

Typology

An arbitrary sum is an endurant.

Description

An arbitrary sum is part an endurant.

Distinctions

No arbitrary sum is a physical endurant, or a non physical endurant.

DEPENDENT PLACE (class)

Typology

A dependent place is a feature.

ENDURANT (class)

Typology

An endurant is a spatio temporal particular.

Description

If any of the following relationships hold between X and Y then Y is an endurant: "is total temporary participant", "is total constant participant", "is temporary proper part of", "is temporary proper part", "is temporary participant", "is temporary part of", "is temporary part", "is temporary atomic part of", "is temporary atomic part", "is participant", "is mereologically coincides", "is life of" and "is constant participant".

If any of the following relationships hold between X and Y then X is an endurant: "is total temporary participant in", "is total constant participant in", "is temporary proper part of", "is temporary proper part", "is temporary participant in", "is temporary part of", "is temporary part", "is temporary atomic part of", "is temporary atomic part", "is participant in", "is mereologically coincides", "is life" and "is constant participant in".

An endurant both is specific constant constituent only an endurant, and is part only an endurant.

Examples

The following are endurants: physical endurants, non physical endurants, and arbitrary sums.

Distinctions

No abstract is an endurant.

No endurant is a quality.

FEATURE (class)

Typology

A feature is a physical endurant.

Description

A feature is host a physical endurant.

If X is host of Y then Y is a feature.

If X is host Y then X is a feature.

Examples

Relevant parts, and dependent places are features.

Distinctions

No amount of matter is a feature.

No feature is a physical object.
