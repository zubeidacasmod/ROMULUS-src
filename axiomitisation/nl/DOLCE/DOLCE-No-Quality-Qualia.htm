ENDURANT (class)

Typology

An endurant is a spatio temporal particular.

Description

If any of the following relationships hold between X and Y then Y is an endurant: "is total temporary participant", "is total constant participant", "is temporary proper part of", "is temporary proper part", "is temporary participant", "is temporary part of", "is temporary part", "is temporary atomic part of", "is temporary atomic part", "is participant", "is mereologically coincides", "is life of" and "is constant participant".

If any of the following relationships hold between X and Y then X is an endurant: "is total temporary participant in", "is total constant participant in", "is temporary proper part of", "is temporary proper part", "is temporary participant in", "is temporary part of", "is temporary part", "is temporary atomic part of", "is temporary atomic part", "is participant in", "is mereologically coincides", "is life" and "is constant participant in".

An endurant both is specific constant constituent only an endurant, and is part only an endurant.

An endurant is participant in a perdurant.

Examples

The following are endurants: physical endurants, non physical endurants, and arbitrary sums.

Distinctions

No endurant is a perdurant.

EVENT (class)

Typology

An event is a perdurant.

Examples

Achievements, and accomplishments are events.

FEATURE (class)

Typology

A feature is a physical endurant.

Description

A feature is host a physical endurant.

If X is host of Y then Y is a feature.

If X is host Y then X is a feature.

Examples

Relevant parts, and dependent places are features.

Distinctions

No amount of matter is a feature.

No feature is a physical object.
