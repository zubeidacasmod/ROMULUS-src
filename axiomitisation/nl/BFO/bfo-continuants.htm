Natural Language View: BFO-Continuants

CONTINUANT (class)

Definition

A continuant is defined as something that is a dependent continuant, is an independent continuant, or is a spatial region.

Typology

A continuant is an entity.

Examples

The following are continuants: spatial regions, independent continuants, and dependent continuants.

DEPENDENT CONTINUANT (class)

Definition

A dependent continuant is defined as something that is a generically dependent continuant, or is a specifically dependent continuant.

Typology

A dependent continuant is a continuant.

Examples

Specifically dependent continuants, and generically dependent continuants are dependent continuants.

Distinctions

No dependent continuant is a spatial region, or an independent continuant.

DISPOSITION (class)

Typology

A disposition is a realizable entity.

Distinctions

No disposition is a role, or a function.

ENTITY (class)

Examples

A continuant is an entity.

FIAT OBJECT PART (class)

Typology

A fiat object part is a material entity.

Distinctions

No fiat object part is any of the following: a site, an object boundary, an object aggregate, or an object.

FUNCTION (class)

Typology

A function is a realizable entity.

Distinctions

No disposition is a function.

No function is a role.

GENERICALLY DEPENDENT CONTINUANT (class)

Typology

A generically dependent continuant is a dependent continuant.

Distinctions

No generically dependent continuant is a specifically dependent continuant.

INDEPENDENT CONTINUANT (class)

Definition

An independent continuant is defined as something that is a material entity, is an object boundary, or is a site.

Typology

An independent continuant is a continuant.

Examples

The following are independent continuants: sites, object boundaries, and material entities.

Distinctions

No dependent continuant is an independent continuant.

No independent continuant is a spatial region.

MATERIAL ENTITY (class)

Definition

A material entity is defined as something that is a fiat object part, is an object, or is an object aggregate.

Typology

A material entity is an independent continuant.

Examples

The following are material entities: object aggregates, objects, and fiat object parts.

Distinctions

No material entity is a site, or an object boundary.

OBJECT (class)

Typology

An object is a material entity.

Distinctions

No fiat object part is an object.

No object is any of the following: a site, an object boundary, or an object aggregate.

OBJECT AGGREGATE (class)

Typology

An object aggregate is a material entity.

Distinctions

No object aggregate is a site, or an object boundary.

No objects or fiat object parts are object aggregates.

OBJECT BOUNDARY (class)

Typology

An object boundary is an independent continuant.

Distinctions

No object boundary is a site.

None of the following are object boundaries: object aggregates, objects, material entities, or fiat object parts.

ONE DIMENSIONAL REGION (class)

Typology

An one dimensional region is a spatial region.

Distinctions

No one dimensional region is any of the following: a zero dimensional region, a two dimensional region, or a three dimensional region.

QUALITY (class)

Typology

A quality is a specifically dependent continuant.

Distinctions

No quality is a realizable entity.

REALIZABLE ENTITY (class)

Typology

A realizable entity is a specifically dependent continuant.

Examples

The following are realizable entities: roles, functions, and dispositions.

Distinctions

No quality is a realizable entity.

ROLE (class)

Typology

A role is a realizable entity.

Distinctions

No functions or dispositions are roles.

SITE (class)

Typology

A site is an independent continuant.

Distinctions

None of the following are sites: object boundaries, object aggregates, objects, material entities, or fiat object parts.

SPATIAL REGION (class)

Definition

A spatial region is defined as something that is an one dimensional region, is a three dimensional region, is a two dimensional region, or is a zero dimensional region.

Typology

A spatial region is a continuant.

Examples

The following are spatial regions: zero dimensional regions, two dimensional regions and three dimensional regions, and so on (4 items in total).

Distinctions

No independent continuants or dependent continuants are spatial regions.

SPECIFICALLY DEPENDENT CONTINUANT (class)

Definition

A specifically dependent continuant is defined as something that is a quality, or is a realizable entity.

Typology

A specifically dependent continuant is a dependent continuant.

Examples

Realizable entities, and qualities are specifically dependent continuants.

Distinctions

No generically dependent continuant is a specifically dependent continuant.

THREE DIMENSIONAL REGION (class)

Typology

A three dimensional region is a spatial region.

Distinctions

No one dimensional region is a three dimensional region.

No three dimensional region is a zero dimensional region, or a two dimensional region.

TWO DIMENSIONAL REGION (class)

Typology

A two dimensional region is a spatial region.

Distinctions

No two dimensional region is a zero dimensional region.

No three dimensional regions or one dimensional regions are two dimensional regions.

ZERO DIMENSIONAL REGION (class)

Typology

A zero dimensional region is a spatial region.

Distinctions

None of the following are zero dimensional regions: two dimensional regions, three dimensional regions, or one dimensional regions.
