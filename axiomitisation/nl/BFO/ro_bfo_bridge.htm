CONTINUANT (class)

Description

If any of the following relationships hold between X and Y then Y is a continuant: "is transformed into", "is transformation of", "is location of", "is located in", "has as participant", "has as agent", "derives from" and "is derived into".

If any of the following relationships hold between X and Y then X is a continuant: "is transformed into", "is transformation of", "participates in", "is location of", "is located in", "derives from", "is derived into" and "is agent in".
