Natural Language View: BFO-Occurents

CONNECTED SPATIOTEMPORAL REGION (class)

Definition

A connected spatiotemporal region is defined as something that is a spatiotemporal instant, or is a spatiotemporal interval.

Typology

A connected spatiotemporal region is a spatiotemporal region.

Examples

Spatiotemporal intervals, and spatiotemporal instants are connected spatiotemporal regions.

Distinctions

No connected spatiotemporal region is a scattered spatiotemporal region.

CONNECTED TEMPORAL REGION (class)

Definition

A connected temporal region is defined as something that is a temporal instant, or is a temporal interval.

Typology

A connected temporal region is a temporal region.

Examples

Temporal intervals, and temporal instants are connected temporal regions.

Distinctions

No connected temporal region is a scattered temporal region.

ENTITY (class)

Examples

An occurrent is an entity.

FIAT PROCESS PART (class)

Typology

A fiat process part is a processual entity.

Distinctions

No fiat process part is any of the following: a processual context, a process boundary, a process aggregate, or a process.

OCCURRENT (class)

Definition

An occurrent is defined as something that is a processual entity, is a spatiotemporal region, or is a temporal region.

Typology

An occurrent is an entity.

Examples

The following are occurrents: temporal regions, spatiotemporal regions, and processual entities.

PROCESS (class)

Typology

A process is a processual entity.

Distinctions

No fiat process part is a process.

No process is any of the following: a processual context, a process boundary, or a process aggregate.

PROCESS AGGREGATE (class)

Typology

A process aggregate is a processual entity.

Distinctions

No process aggregate is a processual context, or a process boundary.

No process or fiat process parts are process aggregates.

PROCESS BOUNDARY (class)

Typology

A process boundary is a processual entity.

Distinctions

No process boundary is a processual context.

None of the following are process boundaries: process aggregates, process, or fiat process parts.

PROCESSUAL CONTEXT (class)

Typology

A processual context is a processual entity.

Distinctions

None of the following are processual contexts: process boundaries, process aggregates, process, or fiat process parts.

PROCESSUAL ENTITY (class)

Definition

A processual entity is defined as something that is a fiat process part, is a process, is a process aggregate, is a process boundary, or is a processual context.

Typology

A processual entity is an occurrent.

Examples

The following are processual entities: processual contexts, process boundaries and process aggregates, and so on (5 items in total).

Distinctions

No processual entity is a temporal region, or a spatiotemporal region.

SPATIOTEMPORAL REGION (class)

Definition

A spatiotemporal region is defined as something that is a connected spatiotemporal region, or is a scattered spatiotemporal region.

Typology

A spatiotemporal region is an occurrent.

Examples

Scattered spatiotemporal regions, and connected spatiotemporal regions are spatiotemporal regions.

Distinctions

No processual entity is a spatiotemporal region.

No spatiotemporal region is a temporal region.

Typology

Distinctions

TEMPORAL REGION (class)

Definition

A temporal region is defined as something that is a connected temporal region, or is a scattered temporal region.
